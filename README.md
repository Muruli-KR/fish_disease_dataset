# Fish Disease Dataset

A categorized image dataset for fish disease detection and classification using Machine Learning / Deep Learning models.

---

## 📂 Dataset Categories

The dataset currently contains the following classes:

- EUS
- Gill Rot
- Healthy Fish
- Red Spot

---

## 🎯 Purpose

This dataset is intended for:

- Fish disease classification
- Deep learning model training
- Computer vision research
- Aquaculture health monitoring systems
- Academic and research projects

---

## 🧠 Possible Use Cases

- CNN-based fish disease prediction
- Transfer learning with TensorFlow/PyTorch
- Image classification projects
- AI-powered aquaculture systems
